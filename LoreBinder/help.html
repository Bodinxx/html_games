Getting Started

  • Security Notice: Initial bootstrap credentials are ADMIN / admin for first access only; rotate them immediately outside development.
  • Use the project toolbar to create, archive, restore, and remove projects.
  • Pick a Primary Theme to seed project CSS defaults, then tailor the overrides to match your setting.
  • Project changes autosave after 5 seconds of inactivity.

Profiles, Preferences, and Password Resets

  • Open Profile to update your shareable user page, website link, and games played.
  • Interface preferences let you adjust the main font size and colour palette.
  • Password reset emails can be requested from the sign-in screen or from your profile security section.
  • Shared profile links use the ?profile=<username> URL parameter.

Editor Modes and Project Structure

  • Visual Layout mode provides a direct editable document pane plus rendered preview.
  • Plain Text mode exposes raw markdown/code editing.
  • Use + Doc and + Folder to build nested chapters.
  • Document tabs let you switch between open files and close them with the × button.

Markdown Features

  • Variables: [var:key="value"] and use with [var:key].
  • Wrapper blocks: ::: wrap-left ... ::: for class-based layout wrappers.
  • Curly blocks: {{monster-stat-block,border-color:crimson ... }}.
  • Links: standard markdown links, local anchors, and doc:<documentId>#anchor.

Admin Console

  • The admin panel now uses Requests, Users, and Projects tabs.
  • Admins can approve account requests, make other users admins, reset user passwords, and ban/delete accounts.
  • The projects tab gives admins a quick view of every project owner, theme, and status.
